# The API

```javascript
import 'gesture-js'
```

## Events

#### major events supported:

* panstart
* panmove
* panend
* swipe
* longpress

#### extra events supported:

* dualtouchstart
* dualtouch
* dualtouchend
* verticalpanstart
* horizontalpanstart
* verticalpanmove
* horizontalpanmove
* tap
* doubletap
* verticalswipe
* horizontalswipe
* pressend
